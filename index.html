Sou Gustavo Merling, um desenvolvedor front-end apaixonado por criar experiências digitais únicas e intuitivas. Com mais de uma década de experiência no mercado, especializei-me em design, marketing digital e na criação de sites institucionais. Meu objetivo é transformar ideias em soluções funcionais e esteticamente agradáveis, sempre com foco na usabilidade e no desempenho.

HTML: Linguagem de marcação para estruturar conteúdo na web.
CSS: Linguagem para estilizar o layout e a aparência dos elementos HTML.
JavaScript: Linguagem de programação para tornar páginas web interativas.
WordPress: Plataforma de gerenciamento de conteúdo para criar sites e blogs.
React: Biblioteca JavaScript para construir interfaces de usuário interativas.
Git: Sistema de controle de versão para gerenciar o histórico de código.
Sass: Pré-processador CSS que adiciona funcionalidades como variáveis e mixins.
Gulp: Ferramenta de automação de tarefas para desenvolvimento web.
npm: Gerenciador de pacotes JavaScript para instalação e gestão de bibliotecas e ferramentas.

Tecnologias de
Desenvolvimento Web

Minha expertise abrange uma ampla variedade de tecnologias que me permitem criar soluções web robustas e eficientes. Tenho uma sólida base em HTML e SASS, que utilizo para estruturar e estilizar sites de maneira eficaz. Para o desenvolvimento de interfaces interativas e dinâmicas, utilizo React, uma biblioteca JavaScript poderosa que possibilita a criação de aplicações rápidas e responsivas.

Além disso, tenho experiência em trabalhar com WordPress para o desenvolvimento de sites e blogs personalizados. Também uso jQuery para simplificar a manipulação do DOM e a interação com o usuário. Ferramentas como Vite.js e Gulp são parte do meu fluxo de trabalho para otimização e automação de tarefas, garantindo uma entrega de projetos mais ágil. Para controle de versão, utilizo Git e GitFlow, assegurando que todo o código seja bem organizado e colaborativo.

Excelência em
Colaboração com Design

Trabalho em estreita colaboração com equipes de UX/UI para garantir que cada projeto seja visualmente atraente e funcional. Embora meu foco principal não seja o design, meu domínio de ferramentas como Figma, Adobe XD e Photoshop me permite entender e implementar com precisão as visões dos designers. Minha atenção meticulosa aos detalhes e meu compromisso com a qualidade asseguram que todos os trabalhos entregues sejam responsivos e ofereçam uma excelente experiência ao usuário, independentemente do dispositivo utilizado.
